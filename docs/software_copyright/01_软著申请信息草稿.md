## 二、软件用途

Sakura 做题集智能错题复盘系统是一套面向个人学习、理工科刷题训练和 AI 辅助复盘的学习管理软件。系统通过 PDF 题本导入、题目拆分、章节识别、错题标注、错因分析、间隔复习、每日练习、AI 学习教练、教材精读和提醒推送等功能，帮助学习者沉淀做过的题目、追踪薄弱知识点、保留多次复盘记录，并基于真实学习证据形成个性化学习建议。

本软件适用于高等数学、计算机专业课程、模拟卷训练、教材精读和其他可通过 PDF 或题图形式管理的学习材料。

## 三、主要功能

1. PDF 题本导入与题目管理  
   支持将 PDF 做题本、模拟卷或教材上传到系统中，按页拆分并保存题图、页码、章节、科目、资料名称等信息。

2. 章节与知识点归类  
   系统优先从页面右上角或题目文本中提取章节信息，清洗重复章节、水印、公众号、基础篇等干扰词，并将题目归入对应知识点或章节。

3. 错题标注与多刷记录  
   学习者可以标记题目的做对、做错、半会、需复习等状态，并选择计算失误、公式遗忘、逻辑死角、题意理解偏差等元认知错因。二刷、三刷或每日练习回填时，系统会按日期保留独立复盘记录。

4. 每日练习与间隔复习  
   系统根据错题、半会题、到期复习题和薄弱章节生成每日练习。复习调度参考 1、3、7、14、30 天间隔，并可按科目、资料、章节和题量进行自定义。

5. AI 学习教练  
   系统支持调用 OpenAI Compatible API，例如 DeepSeek API，用于生成启发式提示、完整解析、举一反三变式、周期反思、教材精读问答和个性化学习规划。

6. 老师记忆与经验归纳  
   系统支持将重要对话或教材解释压缩为老师记忆，并按学科分类、检索和查看。学习者可以自定义记忆压缩规则，使 AI 老师后续回复更贴近个人学习状态。

7. 教材精读  
   系统支持上传教材 PDF，按页保存截图和段落文本。学习者可围绕指定页码或段落向 AI 提问，并可将重要解释压缩导入记忆。

8. 周期反思与统计图表  
   系统支持按周、月生成复盘统计，展示题量、正确率、错因分布、章节正确率和知识点分布，并可调用 AI 生成不足分析和后续规划。

9. 推送与打卡  
   系统支持企业微信机器人、PushPlus 和 SMTP 邮箱推送，可设置早晚提醒、天气提醒、每日打卡入口和推送测试。

10. 数据迁移与安全  
    系统支持本地数据备份、恢复和迁移；公网部署时可设置管理员访问密码、登录失败限速、分级锁定和安全告警。

## 四、技术特点

- 后端采用 Python 标准库 HTTP 服务组织 API，配合 SQLite 保存题库、复习、记忆和设置数据。
- 前端采用原生 HTML、CSS 和 JavaScript 模块化实现，无需 Node 构建流程，便于本地运行和轻量服务器部署。
- PDF 与图像处理使用 PyMuPDF、Pillow 等工具，实现题图渲染、页码导入和教材页处理。
- AI 调用基于 OpenAI Compatible 接口设计，可对接 DeepSeek 或其他兼容模型服务。
- 数学公式渲染使用 MathJax，支持 AI 生成的 LaTeX 公式在网页端展示。
- 推送通道支持企业微信机器人、PushPlus 和 SMTP 邮箱。
- 数据采用本地优先策略，非 AI 功能尽量在本地完成，减少 token 消耗和外部依赖。
