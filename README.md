# Sakura做题集 · 智能错题与备考系统

一个**本地可运行**的做题/错题管理系统：把高数、专业课、英语等 PDF 练习资料拆成可筛选、可标注、可复习的题库，围绕错题做艾宾浩斯复习，并由一个**有持久记忆、越用越懂你的 AI 学习教练**给出量身备考规划。

设计原则：**PDF 导入与分类全部本地完成、零 token；只在你主动点击时才调用大模型。** 不配密钥也能完整体验（自动降级为本地算法）。

---

## 功能总览

### 题库管理
- 上传 PDF，每页自动拆成一道题，保存题图 + 可提取文字；可指定起止页（如只导第 2–10 页）。
- 每道题带**每本从 1 开始的序号**（从第 2 页导入，第一题就是「第 1 题」，与 PDF 绝对页码解耦）。
- 做题本与模拟卷同级管理，各自独立上传；上传后可改名/改科目，不必删重传。
- 本地规则识别章节与知识点（高数/线代/概率/英语/政治…），不耗 token。
- 按科目、做题本、知识点、章节、状态、全文交叉筛选；删空的做题本自动清理。
- **题号快速定位**：筛选到某本/某章后，输入「第几题」即可滚动到该题并高亮（也可在题目详情手填原书印刷题号）。
- 一页多题可用题图裁剪工具处理；前端 MathJax 渲染 LaTeX。
- 题图支持**全屏放大查看**（滚轮缩放 / 拖拽平移 / 双击复位 / Esc 关闭），看题不用再翻书。
- 长列表有**樱花粉滚动条**可拖动，右下角「回到顶部」一键归位。

### 错题与复习
- 标记做对/做错/半会/需复习，记录错因与备注。
- **类艾宾浩斯复习队列**：做错→明天复习，做对后按 1/3/7/14/30 天逐级延长，连续通过标记已掌握。
- 错题须选**元认知错因标签**：计算失误 / 公式遗忘 / 逻辑死角 / 题意理解偏差。
- **每日练习**：从错题、半会、需复习题中按"科目+做题本"分组（每组≤5 道），并给低正确率章节混入约 20% 前置基础题。
- 章节正确率饼图 + 错因雷达图；周/月周期复盘。

### AI 学习教练 · 学习者记忆模型 ⭐
不是每次现算，而是**随你做题不断积累一份会迭代的"学情记忆"**，据此诊断、规划、预测。四层架构（每层固定 schema + 本地 fallback，离线可用）：

| 层 | 作用 | 说明 |
|---|---|---|
| **L0 证据层** | 地面真相 | 做题状态 / 错因标签 / 错题原文 |
| **L1 洞察层** | 抽结构化证据 | 点「错题分析」时，大模型同时返回解析 + 结构化洞察（真实考点、错因枚举、具体误区、缺失前置），写入记忆 |
| **L2 画像层** | 滚动合成档案 | 用拉普拉斯平滑算掌握度、对比上版得趋势，存**版本快照** → 越做版本越高、越准 |
| **L3 决策层** | 诊断/计划/预测 | 查缺补漏排序（每条附可核对证据 + 处方、前置地基优先）、分阶段计划、今日任务、达标预测、班主任寄语 |

进入侧栏「AI 学习教练」：设好每日时间预算与考试日期 → 点「更新学习档案」积累记忆 → 点「生成提分计划」/「让老师点评」。查缺与今日任务都能一键直达对应题库。

pip install -r requirements.txt

## 配置 AI 接口（OpenAI 兼容）

默认接入 **小米 MiMo 开放平台**。在 <https://platform.xiaomimimo.com> 用小米账号登录，控制台创建 API Key，然后：

```powershell
$env:MIMO_API_KEY="你的小米 MiMo Key"
python app.py
```

默认地址 `https://api.xiaomimimo.com/v1`、默认模型 `mimo-v2.5-pro`，无需额外配置即可使用。

### 切换到其它模型 / 端点

代码不绑定任何厂商，所有 OpenAI 兼容端点都能用，通过环境变量切换：

```powershell
# 通用变量（最高优先级）
$env:LLM_API_KEY="..."                              # 密钥
$env:LLM_BASE_URL="https://api.xiaomimimo.com/v1"   # 端点
$env:LLM_MODEL="mimo-v2.5-pro"                       # 模型

# 例：改用 DeepSeek
$env:LLM_API_KEY="你的 DeepSeek Key"
$env:LLM_BASE_URL="https://api.deepseek.com"
$env:LLM_MODEL="deepseek-chat"
```

**密钥优先级**：`LLM_API_KEY` > `MIMO_API_KEY` > `DEEPSEEK_API_KEY`（向后兼容旧配置）。

> 未配置任何密钥时，错题分析、提分计划、复盘等全部自动降级为**本地规则版**，离线可用，只是语言更模板化。

## 微信每日提醒（PushPlus）

每天自动把"今日待复习错题数 + 倒计时 + 各做题本分布"推送到你的微信。

1. 打开 <https://www.pushplus.plus> 用**微信扫码登录**，复制你的 token。
2. 设置环境变量并启动：

```powershell
$env:PUSHPLUS_TOKEN="你的 token"
$env:APP_PUBLIC_URL="http://你的域名或IP:8000"   # 推送正文里的“打开做题集”链接
python app.py
```

3. 手动触发一次试试：`POST /api/push/daily`（成功后微信即收到）。
4. **定时推送**：用 `notify_daily.py` 配合定时器：

```powershell
python notify_daily.py            # 调用运行中的服务推送（推荐）
python notify_daily.py --local    # 服务没在跑时，直接读库推送
```

- **青龙面板**：环境变量加 `PUSHPLUS_TOKEN`，定时任务命令 `task notify_daily.py`，cron 如 `0 8 * * *`（每天 8 点）。
- **系统 crontab / Windows 计划任务**：同样定时执行 `python notify_daily.py` 即可。

> 青龙面板本身是"定时任务平台"，适合跑这个**推送脚本**；做题集 Web 服务本体请用云服务器或内网穿透托管。

## 何时消耗 token

| 阶段 | 是否调用大模型 |
|---|---|
| PDF 导入、拆页、章节/知识点识别、筛选、统计、复习排程 | ❌ 全本地，零 token |
| 读学习档案 / 缓存计划（`GET /api/coach`） | ❌ 零 token |
| 点「错题分析」「Full Solution」「举一反三」「生成总结反思」「更新档案」「生成提分计划」「让老师点评」 | ✅ 调用一次 |

## 技术栈

- **后端** `app.py`：Python 标准库 `http.server` 手写路由 + SQLite，无 Web 框架；`PyMuPDF` 处理 PDF、`Pillow` 裁剪题图；大模型走 OpenAI SDK（兼容端点）。
- **前端** `static/`：原生 HTML/CSS/JS，无构建工具；MathJax 渲染公式，Lucide 图标，精修樱花粉主题。
- **数据**：`documents` / `questions` + 学习者记忆三表 `insights` / `learner_profile` / `coach_state`。
