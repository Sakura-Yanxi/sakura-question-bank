# Sakura 做题集

> 一个面向个人学习与错题复盘的本地优先型智能做题集 demo。  
> 它将 PDF 做题本、模拟卷、教材页与错题复习整合到一个学习面板中，并在需要时调用大模型进行启发式讲解、错题分析、复盘总结和学习计划生成。

---

## 项目定位

Sakura 做题集不是一个商业化题库产品，而是一个围绕“题目不浪费、错题不遗漏、知识点可追踪”的学习系统原型。

它的核心原则是：

- **本地优先**：PDF 拆页、题图保存、题库筛选、错题统计、复习队列都优先在本地完成。
- **AI 按需调用**：只有点击错题分析、Full Solution、AI 学习教练、周期复盘等功能时才调用大模型。
- **数据可迁移**：支持本地与云端之间的数据迁移思路，便于后续从个人 demo 走向长期使用。
- **证据优先**：AI 老师必须优先基于真实做题记录、错题状态、薄弱点排行和复习队列给建议。
- **经验辅助**：可以导入外部经验库，但外部经验只作为参考，不能替代个人真实错题证据。

## 功能概览

### 题库与资料管理

- PDF 做题本导入：按页切分题目，保存题图和可提取文本。
- 模拟卷导入：支持整卷资料与普通做题本分开管理。
- 教材精读：支持上传教材 PDF，按页读取并围绕指定段落向 AI 提问。
- 自定义页码范围：导入时可选择起止页。
- 做题本编辑：导入后可修改名称、科目等信息。
- 题目筛选：按科目、资料、知识点、章节、状态、关键词筛选。
- 题图处理：支持放大查看、缩放、拖拽和边界裁剪。
- 删除与清理：支持删除单题、整套资料；题目清空后自动清理空做题本。

### 错题与复习

- 状态管理：未做、做对、做错、半会、需复习。
- 元认知错因标签：
  - 计算失误
  - 公式遗忘
  - 逻辑死角
  - 题意理解偏差
- 艾宾浩斯式复习队列：做错后进入复习队列，后续做对后按 1 / 3 / 7 / 14 / 30 天延长复习间隔。
- 每日练习：从错题、半会题、到期复习题中生成。
- 自定义每日规则：可按科目、资料、知识点、章节配置推送范围和题量。
- 章节统计：按做题本展示章节正确率、错题分布和错因画像。
- 错题导出：支持按筛选条件导出错题 PDF，便于打印复习。

### AI 学习教练

AI 学习教练目前已经不是单纯的聊天 prompt，而是带有教学策略和学习证据的 tutor agent 雏形。

当前实现包括：

- 意图识别：
  - API 测试
  - 知识讲解
  - 错题复盘
  - 计划制定
  - 节奏调整
  - 记忆更新
- 教学策略选择：
  - 启发式引导
  - 证据诊断
  - 变式训练
  - 复习调度
  - 记忆压缩
- 学习上下文：
  - 老师记忆
  - 学习档案版本
  - 平均掌握度
  - 错因画像
  - 薄弱知识点排行
  - 到期 / 逾期复习量
  - 今日任务
  - 最近错题证据
- 教学回合日志：
  - 记录用户问题
  - 记录识别意图
  - 记录采用策略
  - 记录上下文摘要
  - 记录 AI 回答
  - 生成可压缩进长期记忆的候选内容

### 外部经验库

可以将学长学姐经验、帖子总结、备考方法论、自我反思等内容导入为外部经验库。

经验库字段包括：

- 标题
- 科目
- 标签
- 来源
- 可信度
- 经验正文

注意：外部经验只作为参考。AI 老师引用时必须区分“这是外部经验”与“这是你的真实错题证据”。

### AI 解析与变式

配置大模型 API 后支持：

- Get Hint L1：概念提示
- Get Hint L2：关键步骤提示
- Full Solution：完整 LaTeX 解答
- 举一反三：
  - Base：换数不换逻辑
  - Advanced：变换求解目标
  - Pro：跨章节综合
- 周期反思：
  - 本周复盘
  - 本月复盘
  - 按科目统计做题量、正确量、错题量和薄弱点

### 提醒与推送

- 企业微信机器人推送
- PushPlus 推送
- 每日错题练习推送
- 晚间打卡提醒
- 明天天气推送
- 手机端错题回填链接
- 本地 / 云端打卡模式

## 环境变量配置

建议将配置写入 `.env`，不要写进代码，也不要提交到 GitHub。

### DeepSeek 示例

```env
LLM_API_KEY=你的 DeepSeek Key
LLM_BASE_URL=https://api.deepseek.com
LLM_MODEL=deepseek-chat
```

### 通用 OpenAI Compatible 示例

```env
LLM_API_KEY=你的 API Key
LLM_BASE_URL=https://api.example.com/v1
LLM_MODEL=your-model-name
```

密钥优先级：

```text
LLM_API_KEY > MIMO_API_KEY > DEEPSEEK_API_KEY
```

### 推送配置

```env
APP_PUBLIC_URL=http://你的公网 IP 或域名
WEWORK_BOT_WEBHOOK=企业微信机器人 Webhook
PUSHPLUS_TOKEN=PushPlus Token
```

### 登录保护

部署到公网时建议开启登录保护：

```env
SAKURA_ADMIN_PASSWORD=你的管理员密码
SAKURA_AUTH_SECRET=随机长字符串
```

未配置 `SAKURA_ADMIN_PASSWORD` 时，本地开发模式默认不启用登录保护。

## 云端部署提示

如果部署到中国大陆服务器，域名访问通常需要完成 ICP 备案。否则可能出现“域名暂时无法访问”的服务商拦截页。

可选方案：

- 直接使用服务器 IP 访问。
- 使用已备案域名。
- 使用香港、新加坡、日本等非大陆服务器。
- 通过 Cloudflare 托管 DNS 时，仍需注意源站所在地的备案要求。

详细步骤见：

```text
docs/本地到云端部署保姆级教程.md
```

```text

  -> 错因标注
  -> 间隔复习
  -> 每日练习
```
