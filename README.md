# Sakura 做题集

一个本地运行的做题本 / 错题本管理 demo。它主要做三件事：

- 把 PDF 练习资料拆成题库。
- 记录做题状态、错因、错题复习节奏。
- 基于真实做题记录做统计、筛选和复盘；配置大模型密钥后，再生成更细的错题分析和学习建议。

这个项目的原则是：**导入、拆页、分类、统计、复习排程都在本地完成，不消耗 token；只有你主动点击 AI 相关按钮时才会调用大模型。**

---

## 功能

### 题库与做题本

- 上传 PDF，每页自动拆成一道题，并保存题图和可提取文字。
- 支持导入时自定义起止页，适合只导入某一段练习。
- 每本资料内部按导入顺序生成题号，从第 1 题开始，不依赖 PDF 原始页码。
- 做题本和模拟卷分开管理，各自有独立上传入口。
- 上传后可以编辑做题本名称、科目等信息。
- 按科目、做题本、知识点、章节、状态、关键词筛选题目。
- 删除单题或整套资料；如果某套资料下的题目被删空，会自动清理空做题本。
- 支持题图裁剪，用来处理一页多题或题目边界不准的问题。
- 支持题图全屏查看、缩放、拖拽。

### 错题与复习

- 题目状态支持：未做、做对、做错、半会、需复习。
- 标记错题时可填写备注和错因。
- 错因标签包括：计算失误、公式遗忘、逻辑死角、题意理解偏差。
- 做错后进入复习队列；后续做对会按 1 / 3 / 7 / 14 / 30 天逐级延长复习间隔。
- 每日练习从错题、半会题和需复习题中生成，按科目和做题本分组。
- 章节统计包含正确率、错题分布和错因雷达图。

### 本地学习统计

未配置 API 密钥时，系统仍然可以做本地统计和规划，但它不是 AI 诊断。

本地模式能做：

- 按知识点统计做题数、做对数、做错数和正确率。
- 用平滑后的正确率粗略表示掌握度，避免小样本下出现过于极端的判断。
- 根据正确率、错题数、需复习数和前置知识依赖，给出薄弱点排序。
- 根据剩余天数和每日学习时间，生成一个基础的复习节奏模板。
- 根据复习队列生成今日待复习任务。

本地模式不能做：

- 真正理解题目语义。
- 判断一道题的深层考点。
- 准确识别你具体卡在哪一步。
- 给出可靠的考试水平预测。
- 像老师一样进行个性化诊断。

所以 README 里不再把无密钥模式称为“AI 学习教练”。它更准确的名字是：**本地学习统计与复习规划**。

### AI 功能

配置大模型 API 密钥后，以下按钮才会调用模型：

- 错题分析
- Full Solution
- 举一反三
- 周 / 月总结反思
- 更新学习档案时的语言化诊断
- 生成提分计划或老师点评

AI 模式主要用于：

- 把错题原文、你的备注、错因标签一起交给模型分析。
- 输出更自然的题目解析。
- 抽取更细的错因、缺失前置知识和学习建议。
- 把本地统计结果改写成更像老师反馈的语言。

注意：即使配置了 AI，统计数字仍然来自本地数据库；AI 只负责解释和生成文字，不应该凭空改动你的做题数据。

## 启动

```powershell
pip install -r requirements.txt
python app.py
```

打开：

```text
http://127.0.0.1:8000
```

---

## 配置 AI 接口

默认使用 OpenAI 兼容接口。当前默认端点写的是小米 MiMo，也可以切换到 DeepSeek 或其它兼容服务。

### 小米 MiMo

```powershell
$env:MIMO_API_KEY="你的小米 MiMo Key"
python app.py
```

### DeepSeek

```powershell
$env:LLM_API_KEY="你的 DeepSeek Key"
$env:LLM_BASE_URL="https://api.deepseek.com"
$env:LLM_MODEL="deepseek-chat"
python app.py
```

### 通用配置

```powershell
$env:LLM_API_KEY="..."
$env:LLM_BASE_URL="https://api.xiaomimimo.com/v1"
$env:LLM_MODEL="mimo-v2.5-pro"
python app.py
```

密钥优先级：

```text
LLM_API_KEY > MIMO_API_KEY > DEEPSEEK_API_KEY
```

## 什么时候消耗 token

| 操作 | 是否消耗 token |
| --- | --- |
| PDF 导入、拆页、题图生成 | 否 |
| 本地章节 / 知识点规则分类 | 否 |
| 筛选、统计、错题队列、每日练习 | 否 |
| 查看本地学习统计 | 否 |
| 错题分析、完整解析、举一反三 | 是 |
| AI 总结反思、AI 老师点评、AI 润色计划 | 是 |

## 微信每日提醒

可选功能。配置 PushPlus 后，可以把今日待复习错题数量和倒计时推送到微信。

```powershell
$env:PUSHPLUS_TOKEN="你的 PushPlus token"
$env:APP_PUBLIC_URL="http://你的域名或IP:8000"
python app.py
```

手动触发：

```text
POST /api/push/daily
```

也可以用脚本定时触发：

```powershell
python notify_daily.py
python notify_daily.py --local
```

## 技术栈

- 后端：Python 标准库 `http.server` + SQLite。
- PDF 处理：PyMuPDF。
- 题图裁剪：Pillow。
- 前端：原生 HTML / CSS / JavaScript。
- 公式渲染：MathJax。
- 图标：Lucide。
- AI 调用：OpenAI SDK，支持 OpenAI 兼容端点。

## 当前定位

这是一个学习工具 demo，不是完整商业产品。

它适合用来验证这些想法：

- PDF 做题本自动整理。
- 错题复习队列。
- 做题数据统计。
- AI 对错题分析和复盘的辅助价值。

它不应该夸大成本地 AI，也不应该把简单统计包装成准确预测。真正可靠的部分仍然是：**题目记录、错题回炉、统计追踪和持续复盘**。
